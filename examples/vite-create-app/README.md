# gt-react + Vite Example

A multilingual Vite + React app using `gt-react` for internationalization.

[See it live](https://example-vite-create-app.vercel.app)

## Quick Start

### Clone and install

```bash
git clone https://github.com/generaltranslation/gt.git
cd gt-libraries/examples/vite-create-app
npm install
```

### Run development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
npm run preview
```
