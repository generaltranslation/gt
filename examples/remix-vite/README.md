# gt-react + Remix Example

This example demonstrates how to use `gt-react` with Remix (using Vite) for automatic internationalization.

## Quick Start

### Clone and install

```bash
git clone https://github.com/generaltranslation/gt.git
cd gt/examples/remix-vite
npm install
```

### Run development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
npm start
```
