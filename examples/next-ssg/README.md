# gt-next + Next.js SSG Example

A statically generated multilingual Next.js app using `gt-next` for internationalization.

## Quick Start

### Clone and install

```bash
git clone https://github.com/generaltranslation/gt.git
cd gt-libraries/examples/next-ssg
npm install
```

### Run development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
npm start
```
