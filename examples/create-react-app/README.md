# gt-react + Create React App Example

A multilingual Create React App using `gt-react` for internationalization.

[See it live](https://example-create-react-qhttbgovz-general-translation.vercel.app)

## Quick Start

### Clone and install

```bash
git clone https://github.com/generaltranslation/gt.git
cd gt-libraries/examples/create-react-app
npm install
```

### Run development server

```bash
npm start
```

### Build for production

```bash
npm run build
```
