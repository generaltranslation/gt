# gt-react + Next.js Pages Router Example

A multilingual Next.js app using the Pages Router with `gt-react` for internationalization.

[See it live](https://next-pages-router.vercel.app)

## Quick Start

### Clone and install

```bash
git clone https://github.com/generaltranslation/gt.git
cd gt-libraries/examples/next-pages-router
npm install
```

### Run development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
npm start
```
