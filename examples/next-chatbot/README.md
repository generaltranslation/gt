# gt-next + AI Chatbot Example

A multilingual AI chatbot built with Next.js and `gt-next`, based on the Vercel AI Chatbot template.

[See it live](https://example-ai-chatbot-ten.vercel.app)

## Quick Start

### Clone and install

```bash
git clone https://github.com/generaltranslation/gt.git
cd gt-libraries/examples/next-chatbot
npm install
```

### Run development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
npm start
```
