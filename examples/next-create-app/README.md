# gt-next + Next.js Example

A multilingual Next.js app using `gt-next` for internationalization.

[See it live](https://next-create-app-eight.vercel.app)

## Quick Start

### Clone and install

```bash
git clone https://github.com/generaltranslation/gt.git
cd gt-libraries/examples/next-create-app
npm install
```

### Run development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
npm start
```
