# TanStack Start + General Translation

A minimal example demonstrating internationalization with [gt-tanstack-start](https://www.npmjs.com/package/gt-tanstack-start).

## Quick start

```bash
npm install
npx gt translate   # generate translations (requires GT_API_KEY and GT_PROJECT_ID)
npm run dev
```

Visit [localhost:3000](http://localhost:3000) and use the locale selector to switch languages.
