Initial release of gt-vue, the Vue 3 binding for General Translation: `createGT()` plugin with reactive no-reload locale switching, the `<T>` component with variable and branching components, `useGT`/`useMessages`/`useTranslations` composables, a writable `useLocale()` ref, `<LocaleSelector>`, sync `t()`, and dev hot reload.
