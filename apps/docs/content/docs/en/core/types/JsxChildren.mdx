## Overview

`JsxChildren` represents JSX content that can contain text, elements, and variables for dynamic translation. It's the primary type for structured content that needs to maintain formatting and interactivity across different languages.

```typescript
type JsxChildren = JsxChild | JsxChild[];
```

This type enables rich, translatable content with embedded components, styling, links, and dynamic variables while preserving the structure during translation.

## Type Definition

### JsxChildren Structure

`JsxChildren` can be either a single `JsxChild` or an array of `JsxChild` elements:

```typescript
type JsxChild = string | JsxElement | Variable;
```

| Type | Description | Example |
|------|-------------|---------|
| `string` | Plain text content | `"Hello world"` |
| `JsxElement` | Structured element with tag, props, and children | `{ t: 'div', c: ['content'] }` |
| `Variable` | Dynamic placeholder for runtime values | `{ k: 'userName' }` |

### JsxElement Structure

```typescript
type JsxElement = {
  t?: string;     // tag name
  i?: number;     // id  
  d?: GTProp;     // GT data/properties
  c?: JsxChildren; // children
};
```

### GTProp (GT Properties)

```typescript
type GTProp = {
  b?: Record<string, JsxChildren>; // Branches for conditional content
  t?: 'p' | 'b';                   // Branch transformation type
} & HtmlContentPropKeysRecord;     // HTML attributes (placeholder, title, etc.)
```

## Example

### Basic JSX Content

```typescript copy
import { JsxChildren, Variable } from 'generaltranslation';

// Simple text
const simpleText: JsxChildren = "Welcome to our application!";

// Text with variables
const greeting: JsxChildren = [
  "Hello, ",
  { k: 'userName' } as Variable,
  "! Welcome back."
];

// Mixed content array
const mixedContent: JsxChildren = [
  "You have ",
  { k: 'messageCount', v: 'n' } as Variable,
  " new messages."
];
```

### Structured JSX Elements

```typescript copy
// Simple div element
const divElement: JsxChildren = {
  t: 'div',
  c: ['This is a div with text content']
};

// Element with attributes
const linkElement: JsxChildren = {
  t: 'a',
  d: {
    ti: 'Visit our homepage'  // title attribute
  },
  c: ['Click here to continue']
};

// Nested elements
const nestedContent: JsxChildren = {
  t: 'article',
  c: [
    {
      t: 'h2', 
      c: ['Article Title']
    },
    {
      t: 'p',
      c: [
        'This article was written by ',
        {
          t: 'strong',
          c: [{ k: 'authorName' } as Variable]
        },
        ' on ',
        { k: 'publishDate', v: 'd' } as Variable
      ]
    }
  ]
};
```

### Helper Functions

```typescript copy
// Helper function to create simple elements
function createElement(
  tag: string, 
  children: JsxChildren,
  props?: Partial<GTProp>
): JsxElement {
  return {
    t: tag,
    c: children,
    ...(props && { d: props })
  };
}

// Helper function to wrap text with an element
function wrapText(
  text: string | Variable, 
  tag: string,
  props?: Partial<GTProp>
): JsxElement {
  return createElement(tag, [text], props);
}

// Usage examples
const helpers = {
  boldText: wrapText('Important message', 'strong'),
  linkedText: wrapText(
    'Click here', 
    'a', 
    { ti: 'Navigate to next page' }
  ),
  heading: createElement('h1', ['Page Title']),
  paragraph: createElement('p', [
    'Welcome, ',
    { k: 'userName' } as Variable
  ])
};
```

## Notes

* JSX content maintains structure and formatting during translation
* HTML attributes are preserved using the `d` (data) property with GT-specific keys
* Variables can be embedded anywhere within the content tree
* Elements can be nested to any depth while maintaining translatability
* Accessibility attributes (`aria-label`, `title`, etc.) are fully supported
* The structure is optimized for both translation and rendering performance
