## Overview

The `generaltranslation` library is a TypeScript package for internationalization. It handles translation APIs, locale management, text formatting, and file processing.

Use it standalone or with framework packages like `gt-next` and `gt-react`.

```typescript
import { GT } from 'generaltranslation';

const gt = new GT({
  apiKey: 'your-api-key',
  projectId: 'your-project-id',
  sourceLocale: 'en',
  targetLocale: 'es'
});

// Translate content
const result = await gt.translate('Hello, world!', 'es');
// "Â¡Hola, mundo!"

// Format numbers, dates, currencies
const formattedPrice = gt.formatCurrency(29.99, 'USD');
const formattedDate = gt.formatDateTime(new Date());
// "$29.99"
// "12/25/2023, 3:45:30 PM" (format depends on locale)

// Work with locales
const localeProps = gt.getLocaleProperties('fr-CA');
const isValid = gt.isValidLocale('de');
// { language: "fr", region: "CA", ... }
// true
```

## Configuration

Configuration options via [GTConstructorParams](/docs/core/types/GTConstructorParams):

| Option | Type | Description |
|--------|------|-------------|
| `apiKey` | `string` | Production API key |
| `devApiKey` | `string` | Development API key |
| `projectId` | `string` | Project identifier |
| `sourceLocale` | `string` | Default source language |
| `targetLocale` | `string` | Default target language |
| `locales` | `string[]` | Supported locales list |
| `baseUrl` | `string` | Custom API endpoint |
| `customMapping` | `CustomMapping` | Custom locale definitions |

## Advanced Usage

### Custom Locale Mapping

```typescript
const gt = new GT({
  customMapping: {
    'cn': {
      code: 'zh',
      name: 'Simplified Chinese',
      emoji: 'ðŸ‡¨ðŸ‡³'
    }
  }
});
```

### File Processing Workflows

```typescript
// Upload source files
const uploadResult = await gt.uploadSourceFiles([
  { source: { path: 'src/components/Button.tsx', content: '...' }}
], { sourceLocale: 'en' });

// Enqueue for translation
await gt.enqueueFiles(uploadResult.files, {
  sourceLocale: 'en',
  targetLocales: ['es', 'fr', 'de']
});

// Check translation status
const status = await gt.checkFileTranslations([
  { sourcePath: 'src/components/Button.tsx', locale: 'es' }
]);
```

## Error Handling

Handles errors with descriptive messages:

```typescript
try {
  const result = await gt.translate('Hello', 'invalid-locale');
} catch (error) {
  console.error('Translation failed:', error.message);
}
```
